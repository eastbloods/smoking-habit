# Sigara Takip Uygulaması

Basit ve kullanışlı bir sigara takip uygulaması.

## Özellikler

- Günlük sigara sayısı takibi
- Geçmiş kayıtları görüntüleme
- Günlük, haftalık ve aylık istatistikler
- Mobil uyumlu tasarım

## Kurulum

```bash
npm install
npm start
```

## Teknolojiler

- React
- Material-UI
- React Router
